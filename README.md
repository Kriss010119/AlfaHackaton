# Клонируйте репозиторий
```
git clone https://github.com/Kriss010119/AlfaHackaton.git
cd <имя-папки-проекта>
```

# Перейдите в корневую папку проекта (где находится package.json)
```
cd vite8D
```

# Установите зависимости React приложения
```
npm install
```

# Установите необходимые Python пакеты
```
pip3 install flask flask-cors pandas numpy
```

# Перейдите в папку с Flask приложением (где находится app.py)
```
cd <путь-к-проекту>
```

# Запустите Flask сервер
```
python3 app.py
```

# Вернитесь в корневую папку проекта
```
cd ../ (или cd <путь в корневую папку>)
```

# Запустите React приложение
```
npm run dev
```

Сайт откроется по адресу: http://localhost:5173/
